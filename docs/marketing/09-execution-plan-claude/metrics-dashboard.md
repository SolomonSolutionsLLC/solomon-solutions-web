# Metrics Dashboard — Weekly & Monthly Scorecard
**The numbers that pay, ignore the rest.** Part of [Stage 9](00-master-plan.md). Pulled from the [unified plan §9](../06-unified-marketing-plan-claude.md) and each [funnel's metrics section](../05-funnels-claude/).

> **Tools (all free):** GA4 + Google Search Console (traffic), MailerLite (email), Calendly (calls), App Store Connect (Simply Pray at launch). Track in one simple sheet. **Ignore vanity metrics** — follower counts, impressions, likes. They don't pay. Faith email opens are inflated ~15–20 pts by Apple Mail privacy, so **weight clicks and trends over absolute opens**.

## A. The weekly pulse (5 min, Friday — Agent 10)
Five inputs you control; check momentum, not perfection.

| # | Metric | What good looks like | Where |
|---|---|---|---|
| W1 | **Email subscribers added this week** | trending up week over week (10–25/wk by day 60) | MailerLite |
| W2 | **Warm outreach messages sent** | 5–10/week (Church plan + consulting + HopeStack) | your notes |
| W3 | **Pillar article published?** | 1/week (or 1/2 weeks at 2-hr pace) | Wisdom Walk |
| W4 | **Replies / calls booked** | any reply is a win early; calls = pipeline | inbox + Calendly |
| W5 | **Newsletter sent + open/click** | open 45%+, click 3–6% | MailerLite |

## C. Conversion benchmarks by product (the "is the message working?" table)
When a number sits below the floor, **fix the message before driving more traffic or spending a dollar.**

| Product | Key conversion | Floor → target | Below floor → do this |
|---|---|---|---|
| Simply Pray | landing → email opt-in | 8% → 12–15% | rewrite hero/offer (A/B the 3 heroes) |
| Consulting | landing → booked call | 2% → 5% | sharpen the call's promise; tighten outreach |
| HopeStack | landing → waitlist | 3% → 8% | swap hero to "Protect the people you're trying to help" |
| Heartward | landing → trial start | 2% → 5% | improve the demo block + maker's note |
| Wisdom Walk | landing → subscriber | 15% → 30–40% | fix headline/offer; check sample-quality proof |
| All | trial→paid (Heartward, no-card) | 8% → 15% | improve onboarding + welcome sequence |
| All | email open | 40% → 45%+ | subject lines / list quality / deliverability |

*Sources: [Waitlister](https://waitlister.me/growth-hub/guides/waitlist-landing-page-optimization-guide) · [LanderLab](https://landerlab.io/blog/landing-page-conversion-rate) · [First Page Sage](https://firstpagesage.com/seo-blog/b2b-landing-page-conversion-rates/) · [Powered by Search](https://www.poweredbysearch.com/learn/b2b-saas-landing-page-stats/) · [Adapty](https://adapty.io/blog/trial-conversion-rates-for-in-app-subscriptions/).*

## D. The North Star per product (if you track nothing else)
- **Simply Pray:** waitlist size + **qualified pastor (Church-plan) conversations**.
- **Consulting:** **booked calls/month → clients closed.**
- **HopeStack:** **signed design partners** (3–5 in 90 days).
- **Heartward:** **trial → paid rate** (the number that decides if the business works).
- **Wisdom Walk:** **email signups/week.**

## E. Paid-spend readiness flags (keep $0 until green)
Don't open the ~$1,000/mo paid tap for a product until its three triggers are green — see [master plan §6](00-master-plan.md#6-paid-ad-trigger-gates-per-product--do-not-spend-until-met). Then track **cost per signup / cost per booked call** as the new decisive number; scale only if it beats target, kill if not.
